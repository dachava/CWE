## About the project
Ticketing and platform for CWE, made with Laravel

This is a link to my demo app hosted online, custom made for CWE. Made with Laravel 9, it handles authentication, user verification, roles and permissions, CRUD operations that will allow the modification of front end elements like events, tickets, roster details, etc. By the end of 2022, this app will be installed in a production environment.
